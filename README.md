# 2nd-submission
📌 프로젝트 개요

이 프로젝트는 Python을 활용하여 구현한 콘솔 기반 퀴즈 게임 프로그램이다.
사용자는 메뉴를 통해 퀴즈를 풀고, 새로운 퀴즈를 추가하며, 저장된 퀴즈 목록과 최고 점수를 확인할 수 있다.

프로그램은 단순한 문제 풀이 기능을 넘어, 데이터 저장 및 불러오기 기능(JSON 기반)을 포함하여
프로그램을 종료한 후에도 사용자 데이터(퀴즈, 점수)가 유지되도록 설계되었다.

또한 객체지향 프로그래밍(OOP)을 적용하여 Quiz, QuizGame 등의 클래스를 활용해
기능별로 구조를 분리하고 코드의 재사용성과 가독성을 높였다.

📌 실행 방법 

**1. 시작 (프로그램 시작 흐름)**

```bash
python main.py
```

이 명령어는 `main.py` 파일을 실행하고, 이 파일은 `QuizGame` 클래스를 초기화하여 게임을 시작합니다.
```python
if __name__ == "__main__":
    game = QuizGame()
    game.run()
```

**2. QuizGame 클래스 초기화 (`__init__` 메서드)**

프로그램이 시작되면 `QuizGame` 객체가 생성되고 다음 3가지가 자동으로 실행됩니다:

- `self.quizzes = self.load_quizzes()` → state.json에서 퀴즈 로드 (없으면 기본 퀴즈 로드)
- `self.score = 0` → 현재 점수 초기화
- `self.high_score = self.load_high_score()` → 이전 게임의 최고 점수 로드

**3. 메인 게임 루프 (`run` 메서드)**

프로그램은 무한 루프(`while True`)로 돌아가며, 다음 흐름을 반복합니다:

| 단계 | 설명 |
|------|------|
| **메뉴 표시** | `show_menu()` → 5가지 선택지가 사용자에게 출력됨 |
| **입력 받기** | 사용자가 1~5 중 숫자 입력 |
| **유효성 검증** | 입력값이 1~5 숫자인지 확인 (빈 입력, 문자, 범위 초과 체크) |
| **기능 실행** | 선택 번호에 따라 해당 메서드 실행 |
| **루프 종료** | 5번 선택 시 프로그램 종료, 그 외는 다시 메뉴로 돌아감 |

**4. 각 메뉴별 동작**

| 번호 | 메뉴 | 실행 메서드 | 기능 |
|------|------|-----------|------|
| **1** | 🎯 퀴즈 풀기 | `play()` | 저장된 모든 퀴즈를 순서대로 출제하고 채점, 결과 표시 |
| **2** | ➕ 퀴즈 추가 | `add_quiz()` | 새로운 문제와 선택지를 입력받아 state.json에 저장 |
| **3** | 📋 퀴즈 목록 | `show_quizzes()` | 저장된 모든 퀴즈의 문제만 나열 |
| **4** | 📊 점수 확인 | `show_high_score()` | 현재까지의 최고 점수 표시 |
| **5** | 🚪 종료 | `break` | 프로그램 종료 |

**5. 퀴즈 풀기 상세 과정 (`play` 메서드)**

```
1. 저장된 퀴즈가 없으면 → 경고 메시지 출력 후 반환
2. self.score = 0 으로 초기화
3. 각 퀴즈를 반복 (1번, 2번, 3번...)
   ├─ 문제와 선택지 표시 (quiz.display())
   ├─ 사용자 정답 입력 받기 (get_answer())
   ├─ 정답 확인 (quiz.is_correct())
   │  ├─ 정답 → "✅ 정답입니다!" + score += 1
   │  └─ 오답 → "❌ 오답입니다! 정답은 {번호}번입니다."
   └─ 모든 퀴즈 완료 후 결과 출력 (show_result())
```

**6. 점수 관리 (`show_result` 메서드)**

게임이 끝난 후 다음이 자동으로 실행됩니다:

```
✅ 정답 개수 ÷ 총 문제 수 계산
✅ 새 점수가 최고 점수보다 높으면 → 최고 점수 업데이트 및 state.json 저장
✅ 결과 화면 표시:
   - 총 몇 문제 중 맞혔는지
   - 현재 점수 (점수/총 문제)
   - 최고 점수 (최고점수/총 문제)
```

**7. 데이터 지속성**

프로그램이 종료되어도 데이터는 유지됩니다:

- **`save_quizzes()`** → 새로운 퀴즈 추가 시 state.json에 저장
- **`save_high_score()`** → 최고 점수 갱신 시 state.json에 저장
- **`load_quizzes()`** → 프로그램 시작 시 이전에 저장된 퀴즈 로드
- **`load_high_score()`** → 프로그램 시작 시 이전의 최고 점수 로드



📌 기능 경로 (Function Flow)

프로그램의 각 기능이 어떤 경로로 실행되는지 설명합니다.

**🎯 기능 1: 퀴즈 풀기 (play)**

```
메뉴 선택 (1번)
    ↓
play() 메서드 실행
    ↓
저장된 퀴즈가 있는지 확인?
    ├─ NO → ⚠️ 경고 메시지 출력 후 반환
    └─ YES ↓
현재 점수 초기화 (self.score = 0)
    ↓
각 퀴즈 반복 문제 출력
    ├─ quiz.display()
    ├─ quiz.question, quiz.choices 출력
    ↓
사용자 정답 입력
    ├─ get_answer() 메서드 실행
    ├─ 유효성 검증 (1~4 숫자만)
    ├─ 입력값 반환
    ↓
정답 확인
    ├─ quiz.is_correct(answer) 호출
    ├─ 정답 → print("✅ 정답입니다!") + score += 1
    └─ 오답 → print("❌ 오답입니다! 정답은 {번호}번입니다.")
    ↓
모든 퀴즈 완료
    ↓
show_result() 메서드 실행
    ├─ 점수 계산 (현재 점수 / 총 문제 수)
    ├─ 최고 점수 갱신 여부 확인
    │  ├─ YES → save_high_score() 호출 → state.json 저장
    │  └─ NO → (저장 안 함)
    └─ 결과 화면 출력 (점수, 최고 점수)
    ↓
메뉴로 돌아감
```

---

**➕ 기능 2: 퀴즈 추가 (add_quiz)**

```
메뉴 선택 (2번)
    ↓
add_quiz() 메서드 실행
    ↓
문제 입력 받기
    ├─ get_text_input("문제를 입력하세요: ")
    ├─ 유효성 검증 (빈 입력 제외)
    └─ question 변수에 저장
    ↓
선택지 1~4 입력 받기
    ├─ get_text_input("1번 선택지를 입력하세요: ")
    ├─ get_text_input("2번 선택지를 입력하세요: ")
    ├─ get_text_input("3번 선택지를 입력하세요: ")
    ├─ get_text_input("4번 선택지를 입력하세요: ")
    └─ choice1, choice2, choice3, choice4 변수에 저장
    ↓
정답 번호 입력 받기
    ├─ get_answer_input("정답 번호를 입력하세요 (1~4): ")
    ├─ 유효성 검증 (숫자 여부, 1~4 범위)
    └─ answer 변수에 저장
    ↓
선택지 리스트 생성
    ├─ choices = ["1. {choice1}", "2. {choice2}", "3. {choice3}", "4. {choice4}"]
    └─ 형식으로 정렬
    ↓
Quiz 객체 생성
    ├─ new_quiz = Quiz(question, choices, answer)
    └─ quiz.py의 Quiz 클래스 인스턴스화
    ↓
퀴즈 리스트에 추가
    ├─ self.quizzes.append(new_quiz)
    └─ 메모리의 self.quizzes에 추가
    ↓
JSON 파일에 저장
    ├─ save_quizzes() 메서드 호출
    ├─ state.json 읽기 (high_score 보존)
    ├─ entire_data["quizzes"] 업데이트
    ├─ JSON 형식으로 저장
    └─ print("퀴즈가 저장되었습니다.")
    ↓
메뉴로 돌아감
```

---

**📋 기능 3: 퀴즈 목록 (show_quizzes)**

```
메뉴 선택 (3번)
    ↓
show_quizzes() 메서드 실행
    ↓
저장된 퀴즈가 있는지 확인?
    ├─ NO → ⚠️ "저장된 퀴즈가 없습니다." 출력 후 반환
    └─ YES ↓
"===== 퀴즈 목록 =====" 출력
    ↓
저장된 모든 퀴즈 반복
    ├─ for index, quiz in enumerate(self.quizzes, start=1):
    ├─ print("{index}. {quiz.question}")
    │  (quiz.question만 표시 - 선택지 미표시)
    └─ 모든 퀴즈를 순서대로 출력
    ↓
메뉴로 돌아감
```

---

**📊 기능 4: 점수 확인 (show_high_score)**

```
메뉴 선택 (4번)
    ↓
show_high_score() 메서드 실행
    ↓
등록된 퀴즈 개수 계산
    ├─ total = len(self.quizzes)
    └─ 현재 저장된 퀴즈 수 파악
    ↓
퀴즈가 있는지 확인?
    ├─ NO → ⚠️ "등록된 퀴즈가 없습니다." 출력 후 반환
    └─ YES ↓
잘못된 최고 점수 보정
    ├─ IF self.high_score > total:
    └─ self.high_score = total (부정행위 방지)
    ↓
현재 최고 점수 출력
    ├─ print(f"최고 점수: {self.high_score}/{total}")
    └─ 예: "최고 점수: 4/5"
    ↓
메뉴로 돌아감
```
