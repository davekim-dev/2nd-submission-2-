# 2nd-submission
📌 프로젝트 개요

이 프로젝트는 Python을 활용하여 구현한 콘솔 기반 퀴즈 게임 프로그램이다.
사용자는 메뉴를 통해 퀴즈를 풀고, 새로운 퀴즈를 추가하며, 저장된 퀴즈 목록과 최고 점수를 확인할 수 있다.

프로그램은 단순한 문제 풀이 기능을 넘어, 데이터 저장 및 불러오기 기능(JSON 기반)을 포함하여
프로그램을 종료한 후에도 사용자 데이터(퀴즈, 점수)가 유지되도록 설계되었다.

또한 객체지향 프로그래밍(OOP)을 적용하여 Quiz, QuizGame 등의 클래스를 활용해
기능별로 구조를 분리하고 코드의 재사용성과 가독성을 높였다.
